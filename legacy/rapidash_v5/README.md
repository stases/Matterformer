This is a repository for detailed equivariance study:

 - Clone repo
 - Make environment using env.yaml
 - separate branch for each experiment
 - Make sure to use the same machine for all models for a given experiment for proper comparison -
 - Log every run on wandb
 - Make visualization/plot for each
 - Fill in the tables on overleaf
